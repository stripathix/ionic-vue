# app-vue
I have setup everyting using @sentry/browser. This is Vue + Capacitor + Ionic + @sentry/browser

To run application

1) Go to /app-vue

2) Then run
  ```
    npm install
    npx cap add ios
    npx cap open ios
  ```
3) Add capacitor iOS platform
```
npx cap add ios
```
